# Hierarchical Motion Perception Model

This project explores how to build a hierarchical motion perception model that better approximates the human visual system. The central question is how a model can integrate local image changes into stable object motion and relational structure.

## Research Directions

- Describe motion-signal processing at different levels with a dual-pathway architecture.
- Compare model behavior with human perception in motion integration, material changes, and complex scenes.
- Examine whether deep learning models can acquire more human-like representations when trained under appropriate constraints.

## Expected Contributions

This work aims to provide a computational framework that connects neuroscience, psychophysics, and artificial vision. Instead of evaluating models only by output accuracy, it emphasizes whether their behavior is consistent with human perception.
