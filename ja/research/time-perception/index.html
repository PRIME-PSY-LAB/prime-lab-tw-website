正の IAPS 画像における覚醒度の時間知覚への影響：接近動機の交絡を制御する

時間知覚 感情 接近動機

概要

このプロジェクトでは、正の感情、覚醒度、接近動機が人間の主観的時間をどのように形作るかを検討しました。多くの研究は、高覚醒の感情事象が「長く感じられる」ことを報告していますが、正の感情に関する結果は一貫していません。高覚醒の正の画像が主観的時間を長くする場合もあれば、効果が見られない場合、あるいは短く感じられる場合もあります。

本研究の作業仮説は、この不一致が単なるノイズではなく、接近動機の影響を反映しているというものです。高い接近動機、たとえば報酬を得たいという状態は注意を狭め、時間を短く感じさせる傾向があります。一方、高覚醒は内的時計を速め、時間を長く感じさせる傾向があります。本プロジェクトは Scalar Expectancy Theory (SET) / pacemaker-accumulator model を理論枠組みとして、これらの力がどのように相互作用するかを検証しました。

方法

参加者

三つの実験で、東呉大学の学部生がコンピュータ化された時間推定課題を行いました。各実験の参加者数はおおよそ 30-40 名で、参加者は授業クレジットまたは少額の報酬を受け取りました。

刺激

  • 画像は International Affective Picture System (IAPS) から選択しました。
  • 正の感情価を持つ画像のみを使用し、次の二群に分けました。
    • 低覚醒の正画像
    • 高覚醒の正画像
  • 二群の感情価は対応させ、覚醒度評定のみが信頼できる差を持つようにしました。

課題とデザイン

参加者は、400-1800 ms の八種類の呈示時間のいずれかで画像を観察し、その後、知覚された時間をミリ秒単位で推定しました。主な要因は次の通りです。

  • 覚醒度：高 vs. 低（参加者内）
  • 接近動機：高 vs. 低（参加者間）
  • 呈示順序（参加者間、実験ごとに変更）
    • 実験 1：高覚醒画像と低覚醒画像を完全にランダム混合
    • 実験 2：低覚醒 → 高覚醒のブロック順序
    • 実験 3：高覚醒 → 低覚醒のブロック順序

接近動機は報酬との関連づけによって操作しました。高接近条件では、実験後に一部の画像刺激、または関連するポストカードを報酬として得られる可能性があると伝えました。低接近条件ではその期待を作りませんでしたが、倫理的扱いを揃えるため、すべての参加者に後で小さな贈り物を渡しました。

各参加者・各条件について、物理時間から推定時間を予測する線形回帰を当てはめました。SET の枠組みでは、傾きは pacemaker rate を、切片は switch / attentional gate の有効性を反映します。

主な結果

実験 1：ランダム呈示

高覚醒画像と低覚醒画像をランダムに混合すると、高覚醒画像は一般に低覚醒画像よりも長く判断されました。これは基本的な覚醒効果と一致します。一方で、接近動機の明確な主効果は見られませんでした。

解釈： ランダム混合では、高覚醒試行の影響が感情回復の遅さを通じて隣接する低覚醒試行へ「漏れ出し」、接近動機の役割を覆い隠した可能性があります。

実験 2：低覚醒 → 高覚醒のブロック順序

すべての低覚醒画像を先に呈示し、その後に高覚醒画像を呈示すると、明確な交互作用が現れました。低接近群では高覚醒画像が低覚醒画像よりも長く知覚されましたが、高接近群ではこの覚醒による延長効果が消失しました。平均的には、高接近動機は主観的時間を短くしました。

線形モデル解析では、高接近条件で pacemaker の傾きが低下し、特に高覚醒画像で顕著でした。低接近条件では、高覚醒は主に切片を高めることで主観的時間を長くしました。

解釈： 覚醒が時間的に分離されると、接近動機は覚醒による時間延長効果を打ち消し、有効な pacemaker rate の低下、または pulse loss の増加を通じて作用する可能性があります。

実験 3：高覚醒 → 低覚醒のブロック順序

逆の順序、すなわち高覚醒から低覚醒へ呈示すると、結果パターンは再び変化しました。全体として、高接近動機はより長い主観的時間を導きました。覚醒の主効果は弱く不安定でしたが、時間推定は物理時間とともに増加しました。

解釈： 高覚醒画像を先に呈示することにより、感情調整や注意方略が動員され、後続試行の処理が変化した可能性があります。実験 2 と 3 の対比は、覚醒と動機が時間知覚に与える影響を解釈する際、時間的文脈と感情回復が重要であることを示しています。

統合的結論

  1. 正の覚醒は主観的時間を長くする傾向がありますが、その効果は接近動機と試行順序によって調整されます。
  2. 高い接近動機は、低覚醒試行が先に現れる条件では主観的時間を短くし、別の時間的文脈では pacemaker rate を高めて時間を長くすることもあります。
  3. Scalar Expectancy Theory の枠組みでは、覚醒は主に pacemaker に作用し、接近動機は pacemaker と attentional gate の両方に影響しうると考えられます。
  4. これらの知見は、正の感情研究で覚醒効果が一貫しない理由を説明します。実験ごとに誘発される接近動機の強さや呈示スケジュールが異なるためです。

方法と設備

実験では、ミリ秒単位の刺激呈示が可能なデスクトップコンピュータ、標準的な LCD ディスプレイ、静かな実験室、キーボード入力による数値推定、画像と呈示時間をランダム化するカスタム実験スクリプトを使用しました。
